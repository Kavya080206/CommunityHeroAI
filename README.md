# Community Hero AI 🚨

An AI-powered civic issue reporting platform that enables citizens to report community problems using text, images, videos, and location.

## Features

- AI Issue Analysis (Gemini)
- AI Image Analysis (Gemini Vision)
- Duplicate Report Detection
- Firebase Authentication
- Firestore Database
- Google Maps Integration
- Community Voting
- Admin Dashboard
- Real-time Issue Tracking

## Tech Stack

- React
- TypeScript
- Vite
- Firebase
- Google Gemini AI
- Tailwind CSS
- Google Maps API

## Future Scope

- Video AI Analysis
- Live Notifications
- Role-based Authentication
- Smart Analytics
